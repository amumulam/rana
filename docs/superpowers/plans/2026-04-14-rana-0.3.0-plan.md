# Rana 0.3.0 实现计划 — 输出标准化为字节跳动说明书模板

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 rana 输出从 5 项交付物（A-E）标准化为 9 章节说明书模板，采用四阶段渐进完善流程。

**Architecture:** 单文件增量式输出（`final-analysis.md`），Stage 1 → Basic+Core，Stage 2 → 协作补充，Stage 3 → Detail，Stage 6 程序化验证挂起。

**Tech Stack:** SKILL.md instructions 重写，templates 替换，validator 逻辑调整

---

| 文件 | 改动类型 | 说明 |
|------|---------|------|

| `rana/scripts/quality-validator.py` | 修改 | 验证逻辑改为章节完整性检查（预留，Stage 6 挂起） |
